Read the user's upcoming Google Calendar events.

## When to Use

- User asks "what's on my calendar?" or "what do I have coming up?"
- User wants to check their schedule before planning tasks
- User asks about availability or free time

## Examples

User: "What's on my calendar this week?"
→ read_calendar(days_ahead=7)

User: "Do I have anything tomorrow?"
→ read_calendar(days_ahead=1)

User: "What's my schedule look like?"
→ read_calendar(days_ahead=3)
