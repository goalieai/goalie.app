Mark a specific task as completed.

## When to Use

- User says they finished a task
- User provides an update on a task being done
- User confirms completion of a specific task ID or name

## Examples

User: "Ya terminé de estudiar"  
→ complete_task(task_id="study-task-id")

User: "I've completed my workout"  
→ complete_task(task_name="Workout")
